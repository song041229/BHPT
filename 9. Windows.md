## OS:Windows
- 마이크로소프트 윈도우: 운영체제 집합
  - Windows <n> 엔드 유저 워크스테이션
  - Windows Server NNNN: 서버
- 중요한 이유
  - 전 세계 기업들에서 가장 많이 사용하는 운영체제 및 환경
  - AD/인프라          - EntraID
  - 서버               - Azure
cf) MacOS: 개인용 업무

- 중요한 문제들
  - 워게임, 웹, Pwn, 리버싱 -> Linux 중심
  but,
  - 실제 공격, 네트워크 모의해킹, 인프라, 보안, 호스트 -> Windows 중심
  - 유로 라이센스, 윈도우 보안 접근의 어려움
 
- Windows Shell
  - `cmd` 전통적인 윈도우의 CLI/Shell
  - `Powershell`
    - CLI, 스크립팅 언어, 구성 관리 프레임워크로 구성 (작업 자통화 솔루션)
    - 언어, 쉘, 프레임워크, DLL, .NET
  cf) 윈도우 터미널 `cmd` `Powershell` `Azure Shell` `bash`
