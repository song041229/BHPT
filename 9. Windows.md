### Powershell Command
- `cmdlet (commandlet)` 기본 명령어
- <동사>-<명사> 구성
  ex) `Get-Content` Content를 Get하는 명령어
      `Set-Date` Date를 Set하는 명령어

- `Alias` 명령어 별명
  ex) Get-Content == `cat`
  - 리눅스 사용자들을 겨냥
 
- `스크립트` .ps1 .psd .psm 등등
  - 직접실행
  - 스크립트를 메모리상에 load 후 스크립트 안의 함수 사용
  - In-Memort Execution의 종류


### cmd 실습
- `<명령어> /?` <명령어> 가이드
- `hostname` 현재 <호스트 이름> 출력
- `whoami` 현재 <호스트 이름>/<유저> 출력
- `cd` 현재 디렉토리 출력
  - `cd ..` 상위폴더로 이동
  - `cd .\<디렉토리>` 이동할 <디렉토리>로 이동
- `systeminfo` 호스트와 관련된 정보들을 모두 알려줌
- `cls` clear

* <명령어> /<한 글자 flag>:<인자>
- `dir` 현재 디렉토리 및 하위 폴더 (=ls)
  - `dir /S` hidden file까지 모두 출력
- `echo <문자열>` <문자열> 출력
- `type <파일명>` <파일명> 읽기

- `move <파일명1> <파일명2>` <파일명1>을 <파일명2>로 이름 바꾸기
- `copy <파일명1> <파일명2>` <파일명1>을 <파일명2>에 내용 복사하기
- `del <파일명>` <파일명> 삭제하기
- `mkdir <파일명>` <파일명> 디렉토리 생성

### Powershell 실습
- `Get-Help <명령어>` <명령어>의 Syntax, aliases, remarks 확인
  - `Get-Help -Detailed <명령어>` <명령어>의 Syntax, aliases, parameters, remarks 확인
  - `Get-Help -Examples <명령어>` <명령어>의 예시 확인
- `hostname` 현재 <호스트 이> 출력
- `whoami` 현재 <호스트 이름>/<유저> 출력
- `pwd` 현재 디렉토리 출력
- `cd` 디렉토리 위치
  - `cd ..` 상위폴더로 이동
  - `cd .\<디렉토리>` 이동할 <디렉토리>로 이동
- `Get-Alias <명령어>` <명령어> 별칭이 어떤 명령어인지 알려줌
- `clear` clear

* <명령어> -<단어 flag> <인자>
- `systeminfo` 호스트와 관련된 정보들을 모두 알려줌
  - 중요한 정보 `OS Name` `Version` `System Type` `Hotfix(s)`
- `ls` 권한, 현재 디렉토리 및 하위폴더
  - `ls -Force` hidden flie까지 모두 출력
- `echo "<문자열>"` <문자열> 출력
- `cat .\<파일명>` <파일명> 읽기
  - `cat -First <숫자>.\<파일명>` <파일>에서 첫 줄부터 <숫자>개까지 줄 읽기
  - `cat -Last <숫자>.\<파일명>` <파일>에서 마지막줄부터 거꾸로 <숫자>개까지 줄 읽기

- `>` `>>` `|`
  - `echo "<문자열>" > <파일명>` <파일>의 내용을 <문자열>로 초기화 (redirection)
  - `echo "<문자열>" >> <파일명>` <파일>의 맨 뒤에 <문자열> 추가 (append)
  - `<명령어 문장1> | <명령어 문장2>`
    ex) Get-Process | Where-Object { $_.ProcessName -eq "explorer" }
    - Get-Process의 출력물 중(|), ProcessName이 "explorer"과 동일한(-eq) 한 줄($_.)을 반복({})하여 나타내기
   
- `mv .\<파일명1> <파일명2>` <파일명1>을 <파일명2>로 이름 바꾸기
- `cp .\<파일명1> <파일명2>` <파일명1>을 <파일명2>에 내용 복사하기
- `rm ./<파일명>` <파일명> 삭제하기
- `mkdir <파일명>` <파일명> 디렉토리 생성
