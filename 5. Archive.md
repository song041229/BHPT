## base64
- 민감한 정보를 평문으로 저장하면 보안에 취약함
- base64로 encoding을 이용
- `$base64 <파일이름>` <파일이름>의 내용을 encoding한 결과물 표시
  - `$base64 <파일명1> > <파일명2>` <파일명1>의 내용을 encoding해서 <파일2>에 저장
  - `$base64 -d <파일명>` <파일명>의 내용을 decoding한 결과물 표시

## 압축파일
- 압축파일을 압축해제하여 파일 확인하기
- 확장자 다르면 일치하도록 바꿔주기 (`$file <파일명>`으로 확인, `$mv <파일명> <파일명.확장자>` 이용)
  - `$7z e <파일명>` <파일명> 압축해제 (7-zip archive data)
  - `$unzip <파일명>` <파일명> 압축해제 (Zip archive data)
  - `$gzip -d <파일명>` <파일명 압축해제 (gzip compressed data)
  - `$tar -xvf <파일명>` <파일명> 압축해제 (POSIX tar archive)

## vim
- `$vim <파일명>` 파일이 없다면 생성, 있다면 접속 후 파일 편집
- 모드 종류
  - 노멀 모드
    - `x(커서 뒤)` `X(커서 앞)` 문자 삭제하기
    - `u` 되돌리기
    - `ctrl-r` 다시 실행하기
    - `y$` `p` 커서부터 끝까지 독사하고 붙여넣기
  - `인서트 모드(i)` 텍스트 넣기
  - `커맨드 모드(:)` 명령어 실행
    - `q` vim editer 종료
    - `w` 쓴 내용 저장하기
    - `wq` 저장하고 종료
  - 비주얼 모드(v)
  - 고급 편집
    - `/` 텍스트 검색
    - `:s` 텍스트 치환
    ex. :s/(텍스트검색)/텍스트 치

- 기본이동
  - 커서 이동하기 `h` `j` `k` `l`
  - 줄의 처음/끝으로 이동 `^(공백 포함)` `0(공백 무시, 첫글자부터)` `$`
