## Linux
- 오픈소스 운영 체제 커널
- 서버, 모바일, 워크스테이션, IT, OT, IoT, SCADA 시스템 등 사용

- 공격자들은 왜 Nix (맥, 리눅스, 유닉스)를 많이 쓰나?
  - `오픈소스`&`무료`, 기업/개발자용 운영체제
  - 타겟 OS에 따라 `페이로드 제작`이 용이
  - `풍부한 툴`과 `커뮤니티`

- 왜 Kali Linux를 사용하는가?
  - `다른 보안용 Nix 기반 운영체제` (Arch, Debian, Parrot, RedHat) 존재
  - Offsec 사에서 공식 서포트 (꾸준한 업데이트)
  - 필요한 개발환명&툴 이미 설치 및 apt repo에 존재

## Shell
- 운영체제 <-> 사용자 간의 인터페이스를 제공하는 스프트웨어
- `CLI` Commandline Interface
  - `속도` Text기반 -> 운영체제와 직접적으로 소통(syscalls)
  - `상호운용성` 다른 툴들과의 입/출력 -> 효율적 & 데이터 및 툴 파이프라인 구축 용이
  - `가벼움` 툴 설치 시 많은 좋류의 라이브러리(ex. GUI) 불필요
  - `크로스 플랫폼` 특정 언어 사용 시 다른 웅영체제에서도 실행하기 용이
  - `Headless Environmen` 기업 ro 컨테이너는 GUI X -> 공격자들은 환경 및 대상을 가리지 않고 공
- `GUI` Graphical User Interface
