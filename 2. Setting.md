## Kali Linux
- Kali Linux의 Virtual Machines의 VirtualBox 
- 링크 : https://www.kali.org/get-kali/#kali-virtual-machines
- Setting
  - Taskbar에서 우클릭 -> Panel -> Panel Reference -> Lock panel 해제 후 TaskBar 밑으로 조정 후 다시 체크
  - Applications -> Power Manager -> Display -> `put to sleep bar` `Switch off after` 모두 Never로 바꾸기
  - terminal에서 우클릭 -> Preferences -> Pont 10 -> 14로 변경
  - `$sudo apt update -y` 패키지 업데이트 명령어
  - `$sudo apt install fonts-nanum*` 한클 패키지 설치
  - Oracle VirtualBox 관리자 -> 스냅샷 -> 찍기 : github의 commit와 비슷함

## vpn 실행하기
1. kali linux에서 'Open Terminal Here'로 터미널 실행
2. `$ls` 목록 확인하기 => `~~~.ovpn` 파일 찾기
3. `$sudo openvpn ~~~.ovpn` vpn 실행하기
4. `Initialization Sequence Completed` 메세지 확인
cf. vpn 터미널 끄면 안됨
