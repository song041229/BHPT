## CommandLine
- 읽기 명령어
  - `$ls` 현재 디렉토리 리스트 목록
    - `$ls -l` 파일 속성과 권한 확인하기
    - `$ls -a` hidden 파일(.~~~)까지 모두 출력
  - `$cd <디렉토리>` 현재 디렉토리 위치를 지정한 디렉토리로 바꾸기
    - `$cd ..` : 상위폴더로 이동
    - cf. 파일 이름의 띄어쓰기 처리 `띄어쓰기마다 \`, `앞글자 쓰고 tab`, `" "or ' '이용`
  - `$pwd` 현재 디렉토리 위치 읽기
  - `$cat <파일이름>` 파일 내용 읽기
    - `$cat -n <파일이름>` 줄 번호를 기록하며 파일 읽기
    - `$cat  <파일1> <파일2> > <파일3>` <파일1>과 <파일2> 합쳐서 <파일3에 저장>
  - `$less <파일이름>` 터미널 윈도우 창 자체 스크롤 -> 파일 자체를 스크롤
  - `$head <파일이름>` 파일의 10번째 줄 까지만 출력
    - `$head -n <숫자> <파일이름>` 파일의 <숫자>번째 줄 까지만 출력
  - `$tail <파일이름>` 파일의 뒤에서 10번째 줄 까지만 출력
    - `$tail -n <숫자> <파일이름>` 파일의 뒤에서 <숫자>번째 줄 까지만 출력

- user command 
  - `$ssh <host>@<ip>` host의 계정 접속하기
  - `$hostname` host의 name = ip
  - `whoami` host
  - `$id` 현재 접속한 user의 식별 정보 (UID, GID 등등..)
  - `$uname` 지금 접속한 운영체제 정보
    - `$uname -a` 모든(-a) 정보를 표시
    - `$uname -r` 커널의 release 버전
  - `$man <command>` 메뉴얼
  - `$clear` terminal
