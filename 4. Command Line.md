## CommandLine
- `$ ~~~ | ~~~ ` 파이프라인('|')으로
- `.` 현재 디렉토리
- `..` 한 단계 상위 디렉토리

- 읽기 명령어
  - `$ls` 현재 디렉토리 리스트 목록
    - `$ls -l` 파일 속성과 권한 확인하기
    - `$ls -a` hidden 파일(.~~~)까지 모두 출력
    cf.) `$cat`이용, but Permission denied
      - `$ls -al` 파일 전체 속성 확인
      ex)
      <host>@<hostname>:~$ ls -al
      total 32
      drwxrwx---  4 A user A user 4096 Jul  7 09:32 .
      drwxr-xr-x 15 root   root   4096 Jul  7 09:23 ..
      -rw-r--r--  1 A user A user  220 Jan  6  2022 .bash_logout
      -rw-r--r--  1 A user A user 3771 Jan  6  2022 .bashrc
      drwx------  2 A user A user 4096 Jul  7 09:32 .cache
      -rw-r--r--  1 A user A user  807 Jan  6  2022 .profile
      ---x--x--x  1 A user A user   35 Oct 30  2023 B user-flag.txt
      d---------  2 A user A user 4096 Oct 30  2023 writeToHere

      [분석하기]
      (d or -)(--- --- ---) <숫자> <소유자> <소유그룹> <파일size> ~~~ <name>
      - `d` or `-`
        - `d` 디렉토리
        - `-`디렉토리X
      - `--- --- ---`
        - `rwx`
          - `r` 읽기(read) 권한 = 4
          - `w` 쓰기(write) 권한 = 2
          - `x` 실행(execution) 권한 = 1
          - `-` 해당권한X
        - `---(처음)` <소유자>의 권한
        - `---(중간)` <소유그룹>의 권한
        - `---(끝)` 제 3자의 권한

      1. host와 group 확인하기
      <host>@<hostname>:~$ whoami
      A user
      <host>@<hostname>:~$ groups
      A user
      => 소유자와 소유그룹 모두 일치

      2. 권한 확인하기
      ---x--x--x : 실행 권한만 존재
      
  - `$cd <디렉토리>` 현재 디렉토리 위치를 지정한 디렉토리로 바꾸기
    - `$cd ..` : 상위폴더로 이동
    - cf. 파일 이름의 띄어쓰기 처리 `띄어쓰기마다 \`, `앞글자 쓰고 tab`, `" "or ' '이용`
  - `$pwd` 현재 디렉토리 위치 읽기
  - `$cat <파일이름>` 파일 내용 읽기
    - `$cat -n <파일이름>` 줄 번호를 기록하며 파일 읽기
    - `$cat  <파일1> <파일2> > <파일3>` <파일1>과 <파일2> 합쳐서 <파일3에 저장>
  - `$less <파일이름>` 터미널 윈도우 창 자체 스크롤 -> 파일 자체를 스크롤
  - `$head <파일이름>` 파일의 10번째 줄 까지만 출력
    - `$head -n <숫자> <파일이름>` 파일의 <숫자>번째 줄 까지만 출력
  - `$tail <파일이름>` 파일의 뒤에서 10번째 줄 까지만 출력
    - `$tail -n <숫자> <파일이름>` 파일의 뒤에서 <숫자>번째 줄 까지만 출력
  - `$find` 입력된 조건에 맞는 내용 찾기
    - `$find <디렉토리> <args>`
    - <args> 목록
      - `-type <char>` 찾을 type를 결정 (f: file / ~~~)
      - `-name "<string>"` 찾을 "문자열"을 결정 (*:모든)
      - `-exec <command> ~~~` '~~~ 명령어'를 실행
    - ex) find . -name "user-flag.txt" -exec cp {} . \;
  - `$grep "<string> <파일명>"` <파일명> 안에서 <문자열>로 시작하는 단어 표시 
  - `$file <파일명>` <파일명>의 정보를 보여줌
    ex)
    <host>@<hostname>:~$ file UserB-flag 
    UserB-flag: ELF 64-bit LSB pie executable, x86-64, ~~~~
    - ELF 64-bit LSB pie executable : 64bit 체제의 .exe파일
  - `$strings <파일명>` 읽을 수 있는 문자열들을 표시    
    cf) (응용) `$strings <파일명> | grep "<문자열>"` 읽을 수 있는 string 추출 -> <문자열>시작 표시 
  - `$diff <파일1> <파일2>` 두 파일의 다른 단어 파악

  - 쓰기 명령
    - `$touch <파일명>` 파일 생성
    - `$echo "문자열" > <파일명>` <파일명>에 "문자열" 추가
    - `$cp <파일명1> <파일명2>` <파일명1>의 내용을 <파일명2>로 copy하기
    - `$mv` 파일 이동하기
      - `$mv <파일명1> <파일명2>` <파일명1>의 이름을 <파일명2>로 바꾸기
      - `$mv <파일명> <디렉토리명>` <파일명>을 <디렉토리명>으로 위치 옮기기
    - `$mkdir <디렉토리명>` 새로운 <디렉토리명>의 폴더를 만듬
    - `$rm` 파일 삭제 (보통은 root 권한을 가진 host에서 'sudo rm ~~' 형태로 많이 씀)
      - `$rm -r <디렉토리명>` <디렉토리명>에 해당하는 모든 파일을 -r(재귀적)으로 전부 지우
      - `$rm <파일명>` <파일명> 지우기
    - `$chmod --- <파일명>` <파일명>의 'rwx'권한을 해당하는 숫자에 맞게 변환하기
    - redirector `>` `>>`
      - `$(echo) "<문자열>" > <파일명>` <파일명>에 "문자열"로 초기화하기
      - `$(echo) "<문자열>" >> <파일명>` <파일명>에 "문자열"을 추가하기

- user command 
  - `$ssh <host>@<ip>` host의 계정 접속하기
  - `$hostname` host의 name = ip
  - `$whoami` host
  - `$groups <host>` <host>의 그룹 표
  - `$id` 현재 접속한 user의 식별 정보 (UID, GID 등등..)
  - `$uname` 지금 접속한 운영체제 정보
    - `$uname -a` 모든(-a) 정보를 표시
    - `$uname -r` 커널의 release 버전
  - `$man <command>` 메뉴얼
  - `$clear` terminal
