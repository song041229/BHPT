### Powershell Command
- `cmdlet (commandlet)` 기본 명령어
- <동사>-<명사> 구성
  ex) `Get-Content` Content를 Get하는 명령어
      `Set-Date` Date를 Set하는 명령어

- `Alias` 명령어 별명
  ex) Get-Content == `cat`
  - 리눅스 사용자들을 겨냥
 
- `스크립트` .ps1 .psd .psm 등등
  - 직접실행
  - 스크립트를 메모리상에 load 후 스크립트 안의 함수 사용
  - In-Memort Execution의 종류
