## tmux
- `ctrl + shift + t` 새로운 터미널 열기

### session
- `$tmux ls` 현재 실행중인 세션 목록
- `$tmux new-session -s <세션이름>` <세션이름>의 새로운 세션 생성
- `$tmux a -t <세션이름>`  <세션이름>을 접근(a=attach)하기

### window
- `ctrl + b` tmux prefix key (tmux용 command 사용하기)
  - `(ctrl+b) + c` 새로운 윈도우 생성(create)
  - `(ctrl+b) + n` 다음 윈도우로 이동(next)
  - `(ctrl+b) + p` 전 윈도우로 이동(previous)
  - `(ctrl+b) + <인덱스번호>` 해당 <인덱스>로 이동
  - `(ctrl+b) + ,` 해당 윈도우 이름 바꾸기
  - `(ctrl+b) + `
 
### pane
- `(ctrl+b) + %` 수직분할
- `(ctrl+b) + "` 수평분할
- `(ctrl+b) + <화살표>` 분할된 pane에서 <화살표>방향대로 이동
- `(ctrl+b) + z` 해당 pane 확대하기(zoom) / 원상복귀하기
- `스크롤 모드`
  - `(ctrl+b) + [` 스크롤모드 (우측 상단에 현재line/총line)
  - `PgUp` `PgDn`으로 스크롤 모드에서 페이지 이동
  - `(ctrl+b) + q` 스크롤 모드 나가기(quit)
